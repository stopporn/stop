# 开始

# 1.问题究竟出在哪里

色情/软色情是性欲的催化剂，许多人对性的欲望本来不那么旺盛，是人体的正常水平，完全没有达到不发泄不行的地步，通过意志力的控制或者转移注意力后可以有效的控制，并不会对学习和生活产生很大的影响。但现如今，色情和软色情在网络上泛滥成灾，色情网站屡禁不止，软色情大行其道，人们的欲望被无限激发，人体的繁衍模式被不断唤醒，性变得异常活跃，欲望达到了不发泄不行的地步，越来越多的人通过自慰来发泄欲望，远高于正常水平的刺激让这群人逐渐上了瘾。现在你要还是像之前一样用意志力，或者转移注意力来摆脱这种”瘾”，那显然是不可能的了，想必你也应该试过了。

性功能是人类与生俱来的特性，这本没有错。如果你戒色了很久，始终走不出破戒的怪圈，已经开始对人类天生就具有的性产生了厌恶的情绪，我认为这样是不对的，性是无辜的，性是被胁迫的，它本应该是我们的朋友，是我们身体的组成部分。

性由正常状态逐渐转换为被大量刺激激发的异常状态是问题的所在，让性重回正轨，不再受色情和软色情的危害，不仅需要身体上的努力，更需要心灵上的努力。

# 2.正常模式和繁衍模式

人体在正常模式下，学习和工作有序进行运作，繁衍模式并不会被频繁唤醒，欲望值并不会有很大的累积，通过意志力或者转移注意力对其进行控制的难度还不大。但现在色情和软色情出现了，软色情负责旁敲侧击，对繁衍模式进行初步唤醒，硬色情负责大力助推，直到正常模式被迫切换到繁衍模式。一旦繁衍模式开始运作，所有工作和生活的优先级全部下放，繁衍模式开始主宰一切，就算你第二天有考试，或者工作没做完，在繁衍模式的眼中都不重要。想让繁衍模式切换为正常模式，除了释放欲望的方法之外，几乎是不可能的。

除非说你在看片的时候，有人突然来找你让你出去一趟，这种外力的约束是可以有效地影响繁衍模式。之前看过一些文章，说如果感觉到欲望来袭，那就不要呆在房间内，去室外。但我认为，欲望来袭时，自觉地走出房间的这种内力可能已经不受控制了，或者说呆在房间的趋向力已经大于走出房间的趋向力，所以我认为这种方法对某些人来说是行不通的。

又或者说你在看片的时候突然房子塌了，人体求生的本能会将繁衍模式极速切换至求生模式，来确保人体的安全，那时候如果你还能接着看，接着撸，那我是真的佩服你。

# 3.戒除色情的核心

上文提到想让繁衍模式切换为正常模式，除了释放欲望的方法之外，几乎是不可能的。所以戒除色情的核心就是如何防止繁衍模式被异常唤醒。

繁衍模式的唤醒，分为两个阶段。一个是诱发阶段，一个是激化阶段。诱发因素可能有很多种，比如某些视频，图片，文字，或者现实中的人或物，诱发因素是欲望的导火索，潜移默化的激发你的欲望。正常状态下，人们并不会很顺其自然地，像打开电视剧一样打开硬色情开始看， 而是受诱发因素影响，欲望在你不知不觉中萌发，最终才打开硬色情开始观看，欲望才进入了激化阶段。如果你能罗列出你每次破解的诱发因素，找到这些欲望的导火索，尽量远离这些东西，在欲望强大之前阻断导火索，那戒色也就指日可待了。你仅有的戒色力量能否在欲望强大之前将欲望扼杀在萌发阶段是你戒色能否成功的关键。我们只有将全部的戒色力量都投入到欲望最为薄弱的萌芽期，把力量用在刀刃上，戒色才能成为可能。我们只有先死守住入口，不让更多的刺激涌入进来，内部才有时间进行自我调节，最终才能将欲望的高度活跃的不可控状态逐渐降至平稳的可控状态。

单纯凭借我们现有的力量去警惕那些诱发因素还是不太够的，我们还需要多看些戒色的文章，来对这一力量进行加强。不知道大家认为戒色文章在戒色的生命周期中扮演着什么样的角色，凭什么戒色要去看戒色文章，我为什么要去看戒色文章，我去看戒色文章就能戒掉色了？看戒色文章的意义是什么？看戒色文章的核心逻辑是什么？不知道大家有没有思考过这样的问题。其实戒色说起来很简单，尽量去避开挑起欲望的诱因，不让欲望强大起来就够了。单纯地靠意志力去强戒，就好比欲望开始萌发了，我们自觉的避开了挑起欲望的诱因，但这种自发力是单一的，是无外援的，总有放松警惕或者消耗完的时候，被击垮只是个时间问题。但如果你每天都去看戒色文章，就好比每天都有很多人来告诫你要远离那些挑起你欲望的诱因，要保持清醒，不要被欲望蛊惑。这个时候如果欲望开始萌发了，会有一群人站在你身后为"孤立无援"的你加油助威，可能不需要你使多大劲儿就能避开那些挑起你欲望的诱因，这个时候不需要你刻意地去练习什么念起即断，念起不随，你那时候的所作所为是非常自然的，是顺其自然的，是以压倒性的优势取得胜利的。但现在有些戒色文章还是良莠不齐的，很多说了没说没什么两样，打流水账，甚至找不到戒色的核心，不知道在哪个地方使劲。读者也要学会甄别，不要以为今天看了戒色文章就万事大吉了，自己要学会分析，要结合自身的情况作出改变。

# 4.找回从前的快乐

在现实生活中，大脑认为激发你欲望的东西，和满足你欲望的东西应该是同一个时，大脑才会觉得满足，才会感到快乐。比如一个蛋糕激发了你的食欲，然后你吃到了这个蛋糕，这个时候你才会满足，才会感到快乐。但色情和自慰不是这样的，色情影片中的演员激发了你的欲望，自慰释放了你的欲望，两者并非一个对象，欲望只是得到了释放，并没有得到满足，因此大脑会感到错愕，会感到恍惚，并没有体会到欲望满足后的快乐，反而会觉得在完成一个释放欲望的任务，因为身体欺骗了大脑，大脑是被骑虎难下的身体胁迫了，身体不得不使用自慰的方式释放欲望，以将人体的繁衍模式（高能量消耗）切换到人体的正常模式（低能量消耗）。
